===
ONE looks for TWO in user list and sends him 50
===

Authorization: "Bearer TOKEN_ONE"
POST http://localhost:3000/api/protected/users/list
{
    "filter": "two"
}

POST http://localhost:3000/api/protected/transactions
Authorization: "Bearer TOKEN_ONE"
{
	"name": "Two",
	"amount": 50
}
